# The Star That Couldn't See Its Own Light

Website hadiah interaktif bertema bintang, cahaya, langit malam, dan pengingat lembut bahwa seseorang tetap berharga meski kadang tidak bisa melihat cahayanya sendiri.

## Cara Membuka

Buka `index.html` langsung di browser. Website ini statis dan tidak membutuhkan server.

## Foto Yang Dipakai

Website sekarang memakai dua foto di folder `assets`:

- `assets/776167317077595758.jpg` untuk foto utama di bagian cahaya.
- `assets/Porsuppakarn.jpg` untuk slide lagu "Found" - POR Suppakarn.

Ganti path di `index.html` kalau ingin menukar foto.

## Musik Yang Dipakai

Website sekarang memakai dua musik:

- `assets/the-light.mp3` sebagai musik utama web.
- `assets/found.mp3` untuk slide pesan tersirat "Found".

## Ganti Isi Surat

Isi surat ada langsung di `index.html` pada bagian:

```html
<article class="letter" id="letter">
```

Edit paragraf di dalamnya sesuai pesan pribadi yang ingin diberikan.

## Alur Website

1. Pembuka langit malam.
2. Mini game mencari 5 bintang.
3. Penemuan bahwa cahaya itu tidak pernah pergi.
4. Foto dalam bingkai bercahaya.
5. Bagian "The Light".
6. Slide lagu "Found" sebagai pesan tersirat.
7. Amplop dan surat.
8. Penutup dengan konstelasi dan tombol ulang.

## File Utama

- `index.html` untuk struktur konten, foto, audio, dan surat.
- `styles.css` untuk desain, glow, responsif HP, amplop, dan konstelasi.
- `script.js` untuk transisi halaman, mini game, kontrol musik, dan partikel cahaya.
